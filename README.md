# Welcome to the todo app made by Luki!

A small example app using xstate with remix.

## Stack

- [Remix](https://remix.run)
- [Xstate](https://xstate.js.org/docs/)
- [Tailwind](https://tailwindcss.com/)

## Development

It's pretty simple:

```sh
$ npm install
$ npm run dev
```

You're in business!
